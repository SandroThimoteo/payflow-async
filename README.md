# Async Payment System

Sistema de processamento de pagamentos assíncrono baseado em microsserviços, desenvolvido com .NET, MySQL, AWS e mensageria.

## Stack
- .NET
- ASP.NET Core
- MySQL
- Docker
- AWS SQS / SNS / EventBridge
- Terraform
- xUnit

## Objetivo
Simular um ambiente real de pagamentos com processamento assíncrono, observabilidade, resiliência e deploy em cloud.
